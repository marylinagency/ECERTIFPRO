═══════════════════════════════════════════════════════════════
              الخطوات الفورية (انسخ والصق)
═══════════════════════════════════════════════════════════════

1. تشغيل الـ Migrations:
   npx prisma migrate deploy

2. Seed البيانات:
   npm run db:seed

3. تحقق من الشهادات:
   node scripts/check-certificates.js

4. شغل الخادم:
   npm run dev

═══════════════════════════════════════════════════════════════
              إذا استمرت المشاكل
═══════════════════════════════════════════════════════════════

افتح Developer Console (F12) في المتصفح:
  - ستجد رسائل خطأ تفصيلية مثل:
    [v0] Error loading certificates: ...

في Terminal ستجد:
  [API] Fetching certificates from database...
  [API] Found 4 active certificates

اقرأ: /REPAIRS_SUMMARY.md للتفاصيل الكاملة
