# تشغيل الآن - 3 خطوات فقط

الشهادات الأربع مع الأسعار موجودة وجاهزة! إليك كل ما تحتاجه لتشغيلها:

## 🚀 الخطوة 1: إدراج الشهادات

شغّل هذا الأمر في Terminal:

```bash
npm run db:insert-certs
```

**الحد الأدنى من الوقت: 10 ثوان**

يجب أن تري:
```
✅ تم إنشاء: مبتدئ لغة المرجع - $49
✅ تم إنشاء: مطور لغة المرجع - $99
✅ تم إنشاء: خبير لغة المرجع - $149
✅ تم إنشاء: محترف لغة المرجع المعتمد - $199

✨ اكتمل إدراج الشهادات بنجاح!
```

## 🌐 الخطوة 2: ابدأ الخادم

```bash
npm run dev
```

## 🎯 الخطوة 3: افتح المتصفح

انتقل إلى: `http://localhost:3000`

ستري:
- قسم "شهادات مهنية معتمدة دولياً"
- الشهادات الأربع مع الأسعار
- معلومات عن كل شهادة (المدة، عدد الأسئلة، إلخ)
- كود الكوبون MARJAA100 بارز

## ✅ الشهادات الأربع

| الشهادة | السعر | المستوى | المدة |
|--------|------|--------|-------|
| مبتدئ لغة المرجع | **$49** | مبتدئ | أسبوعين |
| مطور لغة المرجع | **$99** | متوسط | 4 أسابيع |
| خبير لغة المرجع | **$149** | متقدم | 6 أسابيع |
| محترف لغة المرجع | **$199** | خبير | 8 أسابيع |

## 📝 ملاحظات

- ✅ الشهادات مميزة (Featured)
- ✅ أسعار تنافسية
- ✅ معلومات كاملة لكل شهادة
- ✅ عرض ترويجي جاهز (MARJAA100)

## 🆘 إذا لم تظهر الشهادات

1. تأكد من أنك شغلت `npm run db:insert-certs`
2. أعد تحميل الصفحة (F5)
3. تحقق من console في المتصفح (F12) من الأخطاء

## 📚 مراجع إضافية

- اقرأ `INSERT_CERTIFICATES.md` للتفاصيل الكاملة
- اقرأ `CERTIFICATES_STATUS.md` لمعلومات كاملة
- اقرأ `REPAIRS_SUMMARY.md` لجميع الإصلاحات المطبقة
