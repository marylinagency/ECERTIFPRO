# إدراج الشهادات الأربع

إذا كانت الشهادات لا تظهر في الصفحة الرئيسية، اتبع هذه الخطوات:

## الخطوات السريعة

### الطريقة 1: استخدام Script البسيط (الموصى به)

```bash
npm run db:insert-certs
```

هذا سيدرج الشهادات الأربع مباشرة في قاعدة البيانات:

✅ مبتدئ لغة المرجع - **$49** (السعر الأصلي: $99)
✅ مطور لغة المرجع - **$99** (السعر الأصلي: $199)
✅ خبير لغة المرجع - **$149** (السعر الأصلي: $299)
✅ محترف لغة المرجع المعتمد - **$199** (السعر الأصلي: $399)

### الطريقة 2: إعادة تعيين قاعدة البيانات (إذا كنت تريد البدء من جديد)

```bash
npx prisma migrate reset
npm run db:seed
```

## التحقق من نجاح الإدراج

بعد تشغيل الـ script، يجب أن ترى رسالة مثل:

```
🔄 بدء إدراج الشهادات...

✅ تم إنشاء: مبتدئ لغة المرجع - $49
✅ تم إنشاء: مطور لغة المرجع - $99
✅ تم إنشاء: خبير لغة المرجع - $149
✅ تم إنشاء: محترف لغة المرجع المعتمد - $199

✨ اكتمل إدراج الشهادات بنجاح!
```

## تحديث المتصفح

1. تأكد من أن الخادم يعمل:
```bash
npm run dev
```

2. افتح المتصفح على `http://localhost:3000`
3. اذهب إلى قسم الشهادات
4. يجب أن ترى الشهادات الأربع مع أسعارها

## استكشاف الأخطاء

إذا استمرت المشكلة:

1. تأكد من أن DATABASE_URL صحيح:
```bash
echo $DATABASE_URL
```

2. تحقق من اتصال قاعدة البيانات:
```bash
npm run db:test
```

3. جرب Script التحقق:
```bash
node scripts/check-certificates.js
```
